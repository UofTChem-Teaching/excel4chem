5.1 Creating a Scatterplot

The most common type of plot to explore data in the chemistry lab is a scatterplot, which plots each data point independently on the grid space of the graph. To create a scatterplot, select the columns of data you want to visualize and then go to the INSERT ribbon and select CHARTS, then SCATTER, then SCATTER again (this selection may look like a button with a picture of a little scatterplot). Alternatively, you can select the INSERT tab, then CHARTS, then X Y (Scatter). Note that you can select an entire column by clicking on the column heading (i.e. clicking on ‘A’ will select all the data in column ‘A’). If you need to select columns that are not directly adjacent to one another hold the CTRL button (on a Mac hold COMMAND) and click on the column headings you wish to select. If you would like to make a plot with more than I dataset included, as in the example in the gif below where both NO2 and O3 yare plotted on the same timeseries, then you can select all of the columns in question and create the scatter plot as described above.
